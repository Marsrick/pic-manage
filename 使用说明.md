# 私密文件管理系统与漫画阅读器 — 使用说明文档

这是一个为个人隐私设计、具备强 camouflage（伪装）属性的离线文件管理与漫画阅读系统。前端界面采用深色玻璃拟物化设计，完全运行在浏览器沙箱中（离线安全，不依赖后端上传）。

---

## 🚀 1. 本地启动运行

由于系统涉及浏览器原生的 Web Crypto API（用于文件加解密）和 WebAssembly（用于 7z 压缩包解压），浏览器要求必须在 **安全上下文（localhost 或 HTTPS）** 中运行。

### 启动步骤：
1. 双击运行项目目录下的 `server.js`（或在终端执行 `node server.js`）。
2. 在浏览器中访问以下地址（建议在手机或 iPad 浏览器中体验）：
   👉 **[http://localhost:8080](http://localhost:8080)** 👈

---

## 🔒 2. 隐藏入口与管理员模式

应用默认为**普通用户视图**，仅显示公开的文件列表，底部导航也只提供基本的“文件”入口，从外观上看是一个再普通不过的个人相册与文档浏览器。

### 进入管理员模式：
1. 点击顶部标题栏右侧的 **`?`（帮助与反馈）** 按钮，系统会弹出一个普通的反馈表单。
2. 在反馈内容输入框中输入暗号 **`6627`** 并提交。
3. 表单将自动拦截并唤起手势锁验证界面：
   - **首次登录**：需要绘制两次手势密码以初始化管理员凭证。
   - **日常登录**：绘制正确的手势即可解锁管理员模式。
4. 解锁成功后，应用顶部会显示蓝色的“管理员”标章，且底部导航栏会显现出隐藏的 **「管理空间」** 菜单。

### 退出管理员模式：
- 点击顶部标题栏右侧的 **退出图标（带箭头的门）** 即可瞬间切回普通用户模式。
- **后台锁定机制**：将应用切到后台（或锁屏、最小化标签页）时，管理员状态会立即自动失效并切回普通用户状态。返回前台时，会弹出手势验证，验证通过方可继续管理。

## ⚙️ 3. 应用安全设置（普通用户手势锁）

为进一步掩人耳目，普通用户也可以主动启用手势锁。

### 配置说明：
1. 点击顶部标题栏右侧的 **「设置」（齿轮图标）** 按钮。
2. 开启 **“启用启动手势锁”** 开关。
3. 首次开启需绘制两次图形以保存普通用户的解锁手势。

### 🔑 双重手势解锁与分流机制（双密校验）：
启用该功能后，重新打开应用或后台切回前台时，会立刻弹出强制验证手势锁页面，并根据绘制的手势自动分流：
* **绘制管理员手势**：直接解锁并打开 **管理员模式**（自动显示隐藏的管理空间与加密文件）。
* **绘制普通用户手势**：解锁并保持在 **普通用户视图**（仅展示普通文件）。
* **两套手势设置一致**：为防止他人逼迫解锁，若普通手势与管理员手势完全一致，系统默认只解锁并进入 **普通用户视图**，完美保护隐私。
* **绘制错误手势**：直接拦截并提示密码错误。

## 📖 4. 漫画阅读器功能

支持 `.zip` / `.cbz`、`.tar`、`.tar.gz` / `.tgz` 以及 `.7z` 格式漫画，并支持**压缩包套压缩包**的多层嵌套解压。

### 阅读模式：
* **仿真翻页（推荐）**：使用 CSS 3D 转换（`rotateY` 和 `perspective`）与右侧折叠渐变阴影，模拟真实书本的纸页翻动感。
* **左右翻页**：点击屏幕两侧边缘或自动播放进行翻页。
* **上下翻页**：横向平滑滚屏，支持滑动吸附。
* **瀑布模式**：竖向连续长图滚动，适合长条漫画/条漫。

### 性能说明：
- **相邻页面预加载**：系统会在后台静默提前加载并解码后面 2 页及前面 1 页的页面数据，确保 3D 仿真翻页和自动播放时动画如丝般顺滑。
- **自动播放**：控制栏提供“自动翻页”播放器，可自由调节翻页时间间隔。
- **目录侧栏**：点击左侧“目录”可展开抽屉，显示全部页码和章节跳转。

## 📤 5. 文件导入与分类

* **普通用户上传**：点击右下角悬浮上传按钮（FAB），文件将**静默直存为公开模式**（对所有人可见，且没有加密选项，隐藏隐私设定）。
* **管理员上传**：管理员可以通过悬浮按钮或「管理空间」导入文件，并可选择：
  - **加密存储**：使用 PBKDF2 密钥衍生 + AES-256 原生加解密，文件仅管理员自己可见。
  - **公开存储**：文件不加密，普通用户和管理员均可见。

## 📱 6. iOS 客户端打包与安装（Windows 环境）

本项目是一个纯前端静态应用，支持跨平台打包。如果您想将其作为 App 安装在 iPhone/iPad 上，且当前在 Windows 开发环境下，我们为您配置了两种便捷的打包方案：

1. **HBuilderX 云打包（推荐）**：使用内置的 `manifest.json`，在 Windows 上下载 HBuilderX IDE 即可一键发起云打包生成 `.ipa` 安装包。
2. **Capacitor + GitHub Actions（全自动 CI/CD）**：将代码推送到 GitHub，即可利用 GitHub Actions 免费的 macOS 虚拟服务器自动编译出未签名的 `.ipa` 文件。
